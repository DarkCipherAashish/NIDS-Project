# 🛡️ Network Intrusion Detection System (NIDS)

---

## 📌 Project Overview

This project is a Machine Learning-based Network Intrusion Detection System (NIDS) that detects malicious and normal network traffic using the NSL-KDD dataset.

It classifies network activity into:
- Normal traffic  
- Attack types such as DoS, Probe, R2L, and U2R  

---

## 📌 Project Information

This project was assigned and completed as part of the Cyber Security Internship program conducted by Codec Technologies.

It demonstrates the implementation of a Machine Learning-based intrusion detection system for identifying malicious network activity.

---

## 🚀 Features

- Real-time network attack detection  
- Machine Learning model (Random Forest Classifier)  
- Web-based dashboard using Streamlit  
- CSV file upload for predictions  
- Multi-class classification (normal + attack types)  
- High accuracy (~99%)  
- Confusion matrix visualization

## 📊 Dataset

Standard benchmark dataset for intrusion detection systems.

---

## ⚙️ Project Workflow

1. Data loading and exploration  
2. Data preprocessing and cleaning  
3. Feature encoding (categorical → numerical)  
4. Model training using Random Forest Classifier  
5. Model evaluation (accuracy, precision, recall)  
6. Confusion matrix visualization  
7. Deployment using Streamlit  
8. CSV-based prediction system
